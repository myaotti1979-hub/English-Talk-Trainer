📘 English Talk Trainer

AI英会話トレーニングアプリ v2026.05.05a マニュアル

目次 1. 初期設定（APIキー） 2. Free Talk（フリートーク） 3. Flash Translation（瞬間英作文） 4. Shadowing（シャドーイング） 5. 復習システム 6. 単語帳 7. 会話ログ 8. CEFRレベルガイド 9. 学習継続（ストリーク） 10. APIコストについて 11. 効果的な学習法 12. トラブルシューティング

1. 初期設定

必要なAPIキー

Gemini API Key（必須）

会話AI・問題生成・フィードバック評価に使用。

取得先: Google AI Studio

無料ティア: 1日1,500リクエスト（十分な量）

Azure Speech Key（推奨）

音声認識（STT）・音声読み上げ（TTS）・発音評価に使用。

取得先: Azure Portal → Speech Services

Azure Speech リソースを作成し、キーとリージョン（例: eastus）をコピーしてください。

💡 Azure Keyがなくてもテキスト入力だけでFree TalkとFlash Translationは使えますが、音声入力・読み上げ・Shadowingは使えません。

2. Free Talk

AIと自由に英語で会話するモードです。

基本の流れ

  1. カテゴリ（11種）→ トピック（79種）を選択
  2. CEFRレベル（A1〜C1）を選択
  3. クイックモード（3ターン）ON/OFF
  4. 「会話をスタート」をタップ
  5. AIが会話を始める（自動音声読み上げ）
  6. 🎤ボタンで音声入力 or テキスト入力で返答
  7. 「終了」をタップするとフィードバック取得

チャレンジ表現

セッション開始時に、レベルに合った3つの「今日のチャレンジ表現」が提示されます。会話中にこれらを使うことで語彙の幅が広がります。

音声入力

  • 最大録音時間: 60秒
  • 無音が約3秒続くと自動停止
  • 録音中は波形バーが表示されます

AIの声

セッションごとに自動選択されます。

アクセント女性男性出現率
🇺🇸 アメリカJenny, AriaGuy, Davis約60%
🇬🇧 イギリスSoniaRyan約27%
🇦🇺 オーストラリアNatashaWilliam約7%
🇮🇳 インドNeerjaPrabhat約7%

シチュエーション系トピック（ホテル・レストラン等）では場面に合った声が自動選択されます。

フィードバック

会話終了後にAIが4軸で評価：

  • Grammar — 文法の正確さ
  • Vocabulary — 語彙の豊かさ
  • Naturalness — 自然さ
  • CEFR判定 — レベルが適切かの判断

間違いは自動的に「復習キュー」に登録されます。

3. Flash Translation

日本語を見て英語に瞬間変換するトレーニングです。

基本の流れ

  1. カテゴリ（日常会話/ビジネス/旅行/意見・感想）を選択
  2. CEFRレベルを選択
  3. 10問自動生成
  4. 🎤で音声回答 or テキスト入力
  5. AIが採点（0〜100）+ 模範解答 + フィードバック
💡 瞬間英作文は「英語で考える速度」を鍛えます。正確さより「素早く出す」ことを意識してください。

4. Shadowing

AIの英語を聞いて同じ文を繰り返す、発音矯正トレーニングです。

基本の流れ

  1. CEFRレベルを選択
  2. 5文自動生成
  3. 🔊でAIの音声を聞く
  4. 🎤で同じ文を繰り返す
  5. Azure発音評価で採点（正確さ・流暢さ）
⚠️ ShadowingにはAzure Speech Keyが必須です。

5. 復習システム

間違えた表現を間隔反復で復習します。

  • Free Talkの修正点、Flash Translationの不正解が自動登録
  • 復習タイミング: 1日後 → 3日後 → 7日後 → 14日後 → 30日後
  • ホーム画面に復習問題がある場合は「📝 復習問題あり」と表示
  • 「覚えた！」を押すと次の間隔に進む

6. 単語帳

Free Talk終了時に、AIが使ったがユーザーが使わなかった表現を自動抽出して記録します。

  • 英語表現 + 日本語訳 + 会話中の文脈が保存
  • 最大500エントリー
  • ホーム画面の「📚 単語帳」から閲覧

7. 会話ログ

Free Talkの全会話内容が保存されます。

  • 最大50セッション分
  • タップで会話を展開表示
  • 「あの時こう言えばよかった」という振り返りに

8. CEFRレベルガイド

A2 初級〜1,500語過去形・比較級ゆっくり英検3-4級
B1 中級〜3,000語現在完了・条件文普通英検2-準2級

C1 上級〜8,000語+倒置・分裂文速い英検1級

直近10セッションの平均スコアに基づいて、ホーム画面に「おすすめレベル」が自動表示されます。

9. 学習継続（ストリーク）

ホーム画面にカレンダーが表示され、練習した日に色がつきます。連続日数が「🔥 ○日連続」として表示されます。

💡 毎日5分でもいいので続けることが最も重要です。クイックモード（3ターン）なら2〜3分で終わります。

11. 効果的な学習法

🥇 おすすめ日課メニュー（15分）

  1. 📝 復習クイズ（2分）— 前回の間違いを確認
  2. ⚡ Flash Translation 10問（5分）— 変換速度を鍛える
  3. 🗣️ Free Talk クイック3ターン（5分）— 実戦練習
  4. 🎧 Shadowing 5文（3分）— 発音矯正
  • 間違いを恐れない — エラーは学習の材料。AIは judge しません
  • チャレンジ表現を積極的に使う — 得意な言い回しだけで逃げない
  • レベルは少し難しめ — 楽すぎると伸びない
  • 単語帳を定期的にチェック — AIが使った表現を自分のものに
  • 会話ログを振り返る — 「こう言えばよかった」の気づきが宝

12. トラブルシューティング

「Gemini API 400/404」エラー

APIキーが正しいか確認してください。設定画面でキーが見えるので、コピペミスがないかチェック。

「音声認識エラー: Cannot close a closed AudioContext」

v2026.05.04g以降で修正済み。古いキャッシュが残っている場合はブラウザのキャッシュをクリアしてください。

音声が再生されない（iOS）

iOSでは初回に画面タップが必要です。一度どこかをタップしてから音声ボタンを押してください。

マイクが使えない

ブラウザの設定でマイクへのアクセスを許可してください。HTTPSが必要です（GitHub Pagesなら自動的にHTTPS）。

データをリセットしたい

ホーム画面の「🗑️ データクリア」でセッション履歴・ストリーク・単語帳などを一括削除できます。APIキーは保持されます。
