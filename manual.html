1. 初期設定

Gemini API Key（必須） — Google AI Studioで無料取得

Azure Speech Key（推奨） — Azure Portal → Speech Services

💡 Azure Keyがなくてもテキスト入力＋ブラウザ音声で使えます。Shadowing発音評価にはAzure Key必須。

2. 🚀 自動ルーティン（おすすめ）

テーマを選ぶだけで4モードを自動連続。テーマが全セクションで統一されます。

フロー（約15分）：
  1. 🗣️ Free Talk — 3ターン（AIが最後に締める）
  2. ⚡ Flash — 5問（テーマに沿った瞬間英作文）
  3. 🎨 Description — 5シナリオ（段階的に難しくなる描写）
  4. 🎧 Shadowing — 5文（テーマに沿った発音練習）
  5. 🎉 完了 — 4セクション総合スコア表示

12テーマ: 日常/ビジネス/旅行/意見・社会/食べ物/経済・金融/テクノロジー/健康/文化/趣味/家族/自分のこと

3. 🗣️ Free Talk

17カテゴリ・221トピック（うち🎭ロールプレイ68種）。

ターン数選択

3 / 5 / 7 / 10 / ∞ から選択。最終ターンでAIが自然に会話を締めます。

設定AIスタート自分スタート
3ターンAI4回+自分3回自分3回+AI3回
∞ 自由「🏁 終了してフィードバック」で手動終了

🙋 自分からスタート

会議ファシリテーション、プレゼン、道案内など「自分から話し始める」シナリオに最適。

AIの会話スタイル

毎回質問で返さず、共感・自分の話・リアクション・エピソードを自然に混ぜます。

チャレンジ表現

22スキルカテゴリから、その会話トピックで使いやすい短いフレーズ(2-6語)を提案。未使用分は復習キューに自動追加。

⚡ 応答時間計測

フィードバック画面に平均応答秒数を表示。速くなっていく実感が得られます。

4. ⚡ Flash Training

6カテゴリ×CEFR 5段階。毎回30シード×10文体からランダムで40,600+通り。

カテゴリ内容
🏠 日常会話日本語→英語の翻訳練習
💼 ビジネスビジネス場面の翻訳練習
✈️ 旅行旅行場面の翻訳練習
💭 意見・感想意見を英語で表現
⚡ 状況反応状況に英語で即座に反応（翻訳ではなく応答）
🔄 パターンドリル同じパターンを複数状況で反復→チャンク自動化
💡 大文字・句読点は評価対象外（音声入力で制御不能なため）

5. 🎨 Description Training

シナリオベースで「自分の言葉で説明する力」を鍛えるモード。

段階的5問（テーマ×場面×展開 = 3,000+通り）:
  1. 🟢 報告 — 事実を1-2文で伝える
  2. 🟢 叙述 — 出来事を順に2-3文で話す
  3. 🟡 説明 — 原因や理由を2-3文で述べる
  4. 🔴 比較・説得 — 選択肢を比較したり説得する(3-4文)
  5. 🔴 即興対応 — 予想外の展開にその場で対応(3-4文)

例（テーマ:会議 × 場面:上司と一緒 × 展開:時間がない）:

🟢 「上司と会議室に入ったらプロジェクターが壊れていた。受付に状況を伝えて」

🔴 「会議中に突然クライアントから緊急電話。参加者にどう対応するか説明して」

評価: 内容の適切さ・語彙の幅・文法正確さ。模範解答＋✨改善案を表示。

6. 🎧 Shadowing

AIの英語を聞く→繰り返す→Azure発音評価。5文1セット。

  • 🙈 テキスト非表示モード — 耳だけで再現する本格シャドーイング
  • 単語ごと色分け — 🟢正確 🟡要改善 🔴不正確 取消線=未発話
  • 🔊 お手本を聞く — 各文からAI発音を再生（無料）
  • 🎙️ 自分の録音を聞く — AI音声と聞き比べ
  • 🔄 最低スコアをリトライ
  • ✨ AI発音アドバイス（オプション有料）
⚠️ 発音評価にはAzure Speech Key必須

7. 📰 News Discussion

22ジャンルから選択 or フリーワード入力。Gemini Google Search Groundingで最新ニュースを取得し5ターンで議論。ルーティンには含まれません。

8. 復習・フレーズ集・会話ログ

📝 復習

間隔反復（0→1→3→7→14→30日）。フラッシュカード方式。正しい表現を大きく表示＋🔊発音。「😓まだ」で早期再出題、「✅覚えた」で次のインターバルへ。

タイプ出典
🗣️ Free Talk修正フィードバックのcorrections
⚡ Flash不正解Flash評価（日本語元文付き）
💡 未使用チャレンジセッション中に使わなかった表現

📚 フレーズ集

  • 🔊 フレーズの発音再生
  • ✕ 不要フレーズを個別削除
  • ✨ 例文＋🏷️使用シーン生成（有料）

💬 会話ログ

最大50件。🇯🇵和訳表示（有料）で自分の英語が正しく伝わっていたか確認。番号付き1対1対応で翻訳ズレなし。

9. CEFRレベル

レベル語彙文法
A1〜500現在形・canのみ
A2〜1,500過去形・going to・比較級
B1〜3,000現在完了・第一条件文・受動態
B2〜5,000第二第三条件文・過去完了
C1〜8,000+倒置・分裂文・仮定法

TTS速度は全レベル自然速度。レベル差は語彙・文法・AI振る舞いで制御。各レベルにBANNEDリスト付き。

12. 学習のコツ

🚀 ルーティンを毎日

テーマを変えながら朝晩15分。4モード一気通貫でテーマが統一されるので効率的。

📝 復習を忘れずに

間違えた表現が自動で復習キューに。HOMEの復習ボタンを毎日チェック。

📚 フレーズは厳選

知ってるフレーズは✕で消して「使いたい」ものだけ残す。✨で例文＋シーンを生成。

💬 会話ログで振り返り

🇯🇵和訳で「伝えたかったこと」と「実際に伝わったこと」を比較。
