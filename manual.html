1. 初期設定

Gemini API Key（必須） — Google AI Studioで無料取得

Azure Speech Key（推奨） — Azure Portal → Speech Services

💡 Azure Keyがなくてもテキスト入力＋ブラウザ音声で使えます。Shadowing発音評価にはAzure Key必須。

2. 🚀 自動ルーティン

テーマを選ぶだけで4モードを自動連続。テーマが全セクションで統一。約15分。

フロー：
  1. 🗣️ Free Talk — 3ターン（AIが最後に締める）
  2. ⚡ Flash — 5問（テーマに沿った英作文）
  3. 🎨 Description — 5シナリオ（段階的に難しくなる描写）
  4. 🎧 Shadowing — 5文（テーマに沿った発音練習）
  5. 🎉 完了 — 4セクション総合スコア

12テーマ: 日常/ビジネス/旅行/意見・社会/食べ物/経済・金融/テクノロジー/健康/文化/趣味/家族/自分のこと

3. 🗣️ Free Talk

17カテゴリ・221トピック（うち🎭ロールプレイ68種）。

ターン数選択

3 / 5 / 7 / 10 / ∞ から選択。最終ターンでAIが自然に会話を締めます。AIスタートなら自分3回+AI4回、自分スタートなら自分3回+AI3回。

🙋 自分からスタート

会議ファシリテーション、プレゼン、道案内など。

AIの会話スタイル

毎回質問で返さず、共感・自分の話・リアクション・エピソードを自然に混ぜます。

チャレンジ表現

会話トピックで使いやすい短いフレーズ(2-6語)を提案。未使用分は復習キューに自動追加。

4. ⚡ Flash Training

カテゴリ内容
🏠 日常 / 💼 ビジネス / ✈️ 旅行 / 💭 意見日本語→英語の翻訳練習
⚡ 状況反応状況に英語で即反応（翻訳ではなく応答）
🔄 パターンドリル同じパターンを複数状況で反復

30シード×10文体 = 40,600+通りで毎回異なる問題。

5. 🎨 Description Training

具体的なシナリオで「自分の言葉で説明する力」を鍛える独立モード。

段階的5問（テーマ×場面×展開 = 3,000+通り）:
  1. 🟢 報告 — ひとこと〜2文で伝える
  2. 🟢 叙述 — 2〜3文で状況を説明
  3. 🟡 説明 — 理由も含めて説明
  4. 🔴 比較・説得 — 選択肢を比較して意見を言う
  5. 🔴 即興対応 — 予想外の展開に対応

例: 「今朝、駅でコーヒーをこぼしちゃって、隣の外国人のコートを汚してしまった。英語でなんて謝る？」

評価: 内容・語彙の幅・文法。模範解答＋✨改善案を表示。会話ログにも保存。

6. 🎧 Shadowing

  • 🙈 テキスト非表示モード（耳だけで再現）
  • 単語ごと色分け（🟢正確 🟡要改善 🔴不正確 取消線=未発話）
  • 🔊 お手本を聞く / 🎙️ 自分の録音を聞く
  • 🔄 最低スコアをリトライ
  • ✨ AI発音アドバイス（オプション有料）
  • Azure 429自動リトライ（3秒→6秒→9秒）

7. 📰 News Discussion

22ジャンル+フリーワード。ターン数選択(3/5/7)。レベルに応じて記事の長さ・複雑さが変化（A1: 30語, C1: 180語）。ルーティンには含まれません。

8. 復習・フレーズ集・会話ログ

📝 復習

間隔反復（0→1→3→7→14→30日）。フラッシュカード方式。正しい表現を大きく表示＋🔊発音。

📚 フレーズ集

🔊フレーズ再生 / ✕個別削除 / ✨例文＋🏷️使用シーン生成（有料）

💬 会話ログ

最大50件。🇯🇵和訳表示（有料・保存可能 — 次回は無料で即表示）。Description履歴はユーザー発言のみ翻訳。

9. 学習履歴・分析

  • 📊 スキルバランス — モード別レーダー（2週間 vs 1ヶ月）
  • 🎯 弱点分析 — 6軸レーダー（発音/流暢さ/文法/語彙/自然さ/瞬発力）+ 自動アドバイス
  • 📈 スコア推移 — 日別折れ線グラフ（1ヶ月）
  • 📅 週別学習推移 — セッション数＋学習時間バーチャート（8週間）
  • 📋 全セッション一覧

13. 学習のコツ

🚀 ルーティンを毎日 — テーマを変えながら朝晩15分。4モード一気通貫。

📝 復習を忘れずに — 間違い表現が自動で復習キューに。毎日チェック。

📚 フレーズは厳選 — 知ってるものは✕で消して「使いたい」ものだけ残す。

💬 会話ログで振り返り — 和訳で「伝えたかったこと」と「伝わったこと」を比較。

🎯 弱点分析を活用 — 学習履歴のレーダーチャートで弱点を確認し、重点的に練習。
