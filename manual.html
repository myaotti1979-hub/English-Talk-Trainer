📘 English Talk Trainer

AI英会話トレーニング v2026.06.07a

1. 初期設定

Gemini API Key（必須） — Google AI Studioで取得（無料）

Azure Speech Key（推奨） — Azure Portal → Speech Services で作成

💡 Azure Keyがなくてもテキスト入力＋ブラウザ内蔵音声で使えます。Shadowingの発音評価にはAzure Key必須。

2. 🚀 自動ルーティン（おすすめ）

テーマを選ぶだけで3モードを自動連続実行。朝晩10分の学習に最適。

フロー：
  1. HOME画面の「自動ルーティン」をタップ
  2. 12テーマから選択（日常/ビジネス/旅行/食べ物/テクノロジー等）
  3. Free Talk 3ターン → フィードバック確認 → [▶ 次へ]
  4. Flash英作文 10問 → 結果確認 → [▶ 次へ]
  5. Shadowing 5文 → 結果確認 → [🎉 ルーティン完了]
  6. 3セクション総合スコア表示
💡 テーマが全セクションで統一されます。「ビジネス」を選べば、Free Talk→Flash→Shadowingすべてビジネス関連。

3. Free Talk

17カテゴリ・221トピック（うち🎭ロールプレイ68種）。全カテゴリに✏️フリーワード入力対応。

ターン数選択

3 / 5 / 7 / 10 / ∞(自由) から選択。最終ターンでAIが自然に会話を締めます。

設定AIスタート自分スタート
3ターンAI4回 + 自分3回自分3回 + AI3回
∞ 自由「🏁 終了してフィードバック」ボタンで手動終了

🙋 自分からスタート

ONにすると自分が最初に話します。会議ファシリテーション、プレゼン、道案内など「自分から話し始める」シナリオに最適。

AIの会話スタイル

毎回質問で返すのではなく、共感・自分の話・リアクションを自然に混ぜます。尋問ではなく会話になるよう設計。

⚡ 応答時間計測

ターン開始から発話完了までの秒数を自動計測。フィードバック画面で「⚡ 平均応答 X.X秒」と表示。

4. Flash Translation

7カテゴリ×CEFR 5段階。毎回ランダムなサブトピックで多様な問題を生成。

カテゴリ説明
🏠 日常会話日本語→英語の翻訳練習
💼 ビジネスビジネス場面の翻訳練習
✈️ 旅行旅行場面の翻訳練習
💭 意見・感想意見を英語で表現する練習
⚡ 状況反応日本語の状況説明→英語で即座に反応（翻訳ではなく応答）
🔄 パターンドリル同じパターンを3つの状況で反復→チャンクの自動化
🎨 描写トレーニング具体的な場面で自分の言葉で説明。段階的に難しくなる5問（テーマ×場面×展開で3,000+通り）
💡 大文字・句読点は評価対象外（音声入力で制御不能なため）

🎨 描写トレーニング

具体的なシナリオで自分の言葉で説明する練習。DMM英会話のDescriptionレッスンのような形式。

例（テーマ: 職場でのハプニング × 設定: 朝の通勤中 × 展開: 予想外の展開）
  1. 🟢 通勤電車が止まった。上司に状況を簡単に伝えて（1-2文）
  2. 🟡 駅でアナウンスを聞いて代替ルートを探した経験を話して（2-3文）
  3. 🟡 電車遅延が多い原因を説明して（2-3文）
  4. 🔴 バス通勤と電車通勤を比較して同僚に説明して（3-4文）
  5. 🔴 重要な商談の日に電車が止まった！その場でどう対応する？（3-4文）

3,000+通りの組み合わせ（20テーマ×15場面×10展開）で毎回異なる問題。

5. Shadowing

AIの英語を聞く→繰り返す→Azure発音評価でスコア化。5文1セット。

主な機能

  • 🙈 テキスト非表示モード — 録音中は英文をぼかし。耳だけで再現する本格シャドーイング
  • 単語ごと色分け — 🟢正確(80+) 🟡要改善(50-79) 🔴不正確(50未満) 取消線=未発話
  • 🔊 お手本を聞く — 結果画面の各文からAI発音を再生（無料）
  • 🎙️ 自分の録音を聞く — AI音声と聞き比べ
  • 🔄 最低スコアをリトライ — 苦手な文だけ再練習
  • ✨ AI発音アドバイス — 弱点分析＋具体的なコツ（オプション有料）
⚠️ 発音評価にはAzure Speech Key必須。録音開始前にAI音声を自動停止（エコー防止）

6. ニュース議論

7. チャレンジ表現

22スキルカテゴリ×CEFR 5レベル。会話トピックに合った実用的な短いフレーズ（2-6語）を生成。使用判定＋AI評価付き。

💡 未使用のチャレンジ表現は復習キューに自動追加されます

間隔反復（0→1→3→7→14→30日）。即座に復習可能。シンプルなフラッシュカード方式。

📚 フレーズ集

  • ✨ 例文＋使用シーンを生成（オプション有料） — 「🏷️ 週末の予定を話す時」のようなシーンタグ付き
  • 🔊 例文の音声再生

💬 会話ログ

  • 最大50セッション保存。タップで展開
  • 🇯🇵 和訳を表示（オプション有料） — 自分の英語が正しく伝わっていたか確認

9. CEFRレベル

レベル語彙文法目安

A2〜1,500過去形・going to・比較級英検3-4級

12. 学習法

🥇 自動ルーティンを使おう（10分）

HOME画面の「自動ルーティン」をタップ → テーマを選ぶだけ。Free Talk → Flash → Shadowingを自動で一気通貫。テーマが統一されるので学習効果◎。

📝 復習を毎日チェック

間違えた表現が自動で復習キューに。忘れそうなタイミングで出題（間隔反復）。

📚 フレーズ集は厳選する

自動抽出された中から「知っている」ものは✕で消して、「使いたい」フレーズだけ残す。✨で例文＋使用シーンを生成して🔊で繰り返す。

💬 会話ログで振り返り

🇯🇵和訳表示で「自分が言いたかったこと」と「実際に伝わったこと」を比較。

13. トラブルシューティング

マイクが使えない

PWAモード問題: アプリアンインストール → Chromeで開き直す → マイク許可 → 再インストール。または⚙️設定で「Azure STT」に切替。

404エラー / モデルが見つからない

廃止されたモデルが保存されている可能性。⚙️設定で「3.1 Flash-Lite」を選び直す。アプリは起動時に廃止モデルを自動検出・リセットします。

503エラー

自動リトライ（2秒後1回）あり。頻発なら⚙️設定で別のモデルに切替。

Shadowingでスコアが出ない

Azure Speech Key必須。「認識」テキストが表示されればAzureは動作中。スコアが出ない場合はリトライ。
